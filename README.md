# Qureka : 자기주도 학습을 하는 대학생을 위한, 강의자료를 분석하고 개인 맞춤형 질문을 생성하여 깊이 있는 학습을 유도하는 AI 튜터링 서비스
---
## 서비스 개요
PDF 강의록, 학생의 손필기, 강의 녹음 파일에 대한 종합적인 분석을 기반으로 비판적 사고 질문과 소크라테스식 대화를 생성하여, 학생의 깊이 있는 학습을 돕는 능동적 학습 파트너로서 기능하는 AI 조교 웹 서비스이다.

---
## 주요 기능
1. PDF 강의록, 내 손필기, 강의 녹음 파일을 한곳에서 보기
2. 핵심 주제에 대한 이해도 점검을 위한 비판적 질문 던지기
3. 사용자가 질문에 답하고 궁금한 점을 다시 질문하며 능동적 학습 돕기

---
## 적용 기술
1. FrontEnd
- React.js : 사용자 인터페이스(UI) 개발, 대화 화면, 자료 업로드/조회 기능 구현
3. BackEnd
- FastAPI : API 서버, AI 모델 연동, 데이터 처리 및 LLM 호출 관리
4. DataBase 
- PostgreSQL : 사용자 계정, 대화 로그, 업로드 자료 메타데이터 관리
- pgvector :  임베딩 벡터 저장 및 유사도 검색, AI 질문 추천 및 검색 최적화
5. AI 
- OpenAI Whisper : 강의 음성 파일(STT) → 텍스트 변환
-	PyMuPDF	PDF : 강의 파일에서 텍스트 추출
-	EasyOCR	PDF : 강의록과 필기 이미지에서 텍스트 추출 (OCR)
-	LangChain	: 여러 데이터 소스 연동, LLM 체인 구성, 질문 생성 파이프라인 관리
-	OpenAI : 소크라테스식 질문 생성, 학생 답변 분석, 맞춤형 피드백 생성

## 핵심 기술 요소 
1. RAG
- '강의 자료'의 벡터화 및 인덱싱 (Indexing) : 업로드된 강의 자료(PDF, 음성 등)를 LLM이 참조할 수 있도록 검색 가능한 '의미 벡터'로 변환 및 저장
- 질문을 위한 컨텍스트 검색 (Retrieval) : AI가 선제적 질문을 생성하거나 사용자의 답변에 꼬리를 물기 위해, 전체 강의 자료 중 가장 관련성 높은 텍스트 조각(Context)을 실시간으로 검색
- '강의 자료' 기반의 질문 생성 (Grounded Generation) : LLM의 일반 지식으로 질문의 깊이를 더하되, 내용은 '강의 자료'의 맥락에 철저히 기반한 소크라테스식 질문 생성

2. 프롬프트 엔지니어링
- 컨텍스트 기반 질문 생성을 위한 프롬프트 구조화 : 추출된 강의 원문 텍스트를 LLM에 효과적으로 전달하기 위한 최적의 프롬프트 템플릿 설계
- LLM 최적화를 위한 프롬프트 설계 : 강의 내용에 기반하여 고품질의 질문을 생성하도록 프롬프트 템플릿 연구 및 설계
- 대화 맥락 유지를 통한 연속적 질문 생성 : 사용자와의 이전 대화를 데이터베이스에 기록하고, 이를 다음 프롬프트에 맥락 정보로 함께 포함하여 더 깊이 있는 상호작용 유도
