# 👩‍🏫자기주도 학습을 하는 대학생을 위한, 강의자료를 분석하고 개인 맞춤형 질문을 생성하여 깊이 있는 학습을 유도하는 AI 튜터링 서비스
---
## 서비스 개요
PDF 강의록, 학생의 손필기, 강의 녹음 파일에 대한 종합적인 분석을 기반으로 비판적 사고 질문과 소크라테스식 대화를 생성하여, 학생의 깊이 있는 학습을 돕는 능동적 학습 파트너로서 기능하는 AI 조교 웹 서비스이다.

---
## 주요 기능
1. PDF 강의록, 내 손필기, 강의 녹음 파일을 한곳에서 보기
2. 핵심 주제에 대한 이해도 점검을 위한 비판적 질문 던지기
3. 사용자가 질문에 답하고 궁금한 점을 다시 질문하며 능동적 학습 돕기

---
## 적용 기술
STT, OCR, LLM 등
